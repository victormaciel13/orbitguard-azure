Propósito

Reduzir o tempo entre a captura de um sinal ambiental no espaço e a tomada de decisão em solo, usando uma infraestrutura cloud segura, automatizada e monitorada.

01

O problema espacial que resolvemos

Queimadas e enchentes evoluem em horas, mas os dados de observação da Terra ficam dispersos em diferentes provedores orbitais, em formatos pesados e sem tratamento. Quando a informação chega à defesa civil, muitas vezes já é tarde.

O OrbitGuard AI atua exatamente nesse intervalo: ingestão contínua de imagens de satélite, classificação por visão computacional (cobertura do solo, foco de calor, corpos d'água) e emissão automática de alertas de risco — tudo em uma infraestrutura cloud elástica que escala com o volume orbital.

Como a solução funciona

🛰️

1. Ingestão orbital

Um Timer Trigger dispara periodicamente uma Azure Function que simula a coleta de um novo tile de satélite.

→
🧠

2. Análise por IA

A Azure Function classifica a cobertura do solo (o modelo CNN da entrega de Computer Vision) e calcula um índice de risco ambiental.

→
🚨

3. Alerta de risco

Alertas são persistidos no Cosmos DB e expostos via HTTP Trigger, consumidos por este painel servido pelo Azure Static Web Apps.

03

Painel ao vivo de alertas orbitais

Dados servidos pelo backend serverless (Azure Functions + Cosmos DB). Se a API ainda não estiver publicada, o painel exibe dados de demonstração.
